Fibonacci Brainteaser for HTML and CSS

Can you make images s1 - s9 into a Fibonacci spiral using only HTML and CSS?

Rules

  1. You cannot add HTML.
  2. You can change the order of the images in HTML.
  3. You can add styles. (e.g. padding, margin, z-index, float)
  4. You cannot use absolute positioning.
  5. You cannot use tables. (Duh.)
